# YTX 物品管理

大家好，今天要给大家详细介绍一下 **YTX** 是如何管理物品的。

本次演示的软件版本是 **0.2.6**。

---

## 主界面

首先，我们来看一下主界面。

1. **登录**
   - 登录时需要输入 **邮箱**、**密码** 和 **工作区**。
   - 邮箱和密码是用户自己注册的，而工作区是由管理员在后台创建的，用户需要申请加入某个工作区，才能获得相应的访问权限。
   - 如果你经常使用同一台电脑，可以选择 **本地保存密码**，下次就能直接登录了。

2. **菜单**
   - 主界面顶部是 **菜单栏**，包括：文件、编辑、账号。
   - 下面是 **工具栏**，提供常用功能：插入节点、追加节点、编辑节点名称、重置节点颜色、创建节点组、追加记录、删除、全选、反选、空选、账单、结算。
   - 左侧是 **版块栏**，目前有四个：财务、物品、人脉和任务。

3. **物品版块**
   - 在不同的场景中，有人习惯叫它 **物品**，也有人称作 **产品** 或 **仓库**。在 YTX 中，这些都统一称为 **物品**。
   - 初始情况下，这里是空的。需要通过 **节点** 来逐步搭建结构。

---

## 核心属性

接下来，我们要理解节点的 **三个核心属性**。

- 第一个是 **规则（Rule）**。
  节点可以设置为 `DICD` 或 `DDCI`，这会直接决定叶节点的数据如何计算。

- 第二个是 **类型（Type）**。
  - **B，也就是枝节点**，它的作用是对下级叶节点进行汇总。
  - **L，也就是叶节点**，这是最终输入数据的地方。

- 第三个是 **单位（Unit）**。
  单位有三种：
  - **INT，内部编号**，也就是系统自己的编号。
  - **EXT，外部编号**，通常用来做映射。
  - **POS，位置**，表示存放的位置。
  在实际使用中，内部编号和位置最常用，外部编号则用来做关系映射。

## 演示

下面我们通过一个小例子，来看看物品版块是如何使用的。

### 节点操作

1. **添加一个产品枝节点**
   - 快捷键：`Alt + P`
   - 我们命名为 `苹果`，类型选择 `B`，单位选择 `INT`。

2. **在 `苹果` 下添加三个叶节点**
   - 选中 `苹果`，再按 `Alt + P`。
   - 分别命名为：`手机`、`电脑` 和 `平板`，类型为 `L`，单位同样是 `INT`。

3. **在 `苹果` 下添加一个位置枝节点**
   - 选中 `苹果`，然后按 `Alt + N`，这是同层插入。
   - 命名为 `位置`，类型为 `B`，单位设置为 `POS`。

4. **在 `位置` 下再添加三个叶节点**
   - 选中 `位置`，按 `Alt + P`。
   - 分别命名为：`1号箱`、`2号箱` 和 `3号箱`，类型为 `L`，单位是 `POS`。

到这里，我们的结构就完成了。你可以自由展开或收起枝节点，也可以拖动节点调整位置。

**需要注意的几点：**

- 使用 `Alt + P` 时，会在当前枝节点下追加子节点；如果没有选中节点，就会追加到根节点，也就是顶级节点。
- 使用 `Alt + N` 时，会在当前节点的同一层级插入新节点；如果没有选中节点，就不会有动作。
- 节点的 **类型和单位一旦设定，就不能修改**。如果填错了，只能删除重新建。
- 枝节点只会对 **相同单位** 的叶节点进行汇总。
- 特别提醒：`位置` 节点的单位必须准确标识。在订单版块开单时，系统不会收集位置节点的数据。

### 记录操作

接下来是记录操作。

1. **添加记录**
   - 双击 `苹果 - 手机`，就能进入记录页面。
   - 此时工具栏会自动高亮可用的功能。

2. **记录的关键属性**
   - **D**：用于添加相关文档。这里仅保存路径，不会存储实际文件。
   - **C**：用于盘点。
   - **关联**：可以关联到 **位置** 或 **其他产品**。
     - 如果关联位置，表示存放地点。
     - 如果关联产品，表示下一步生产环节。
   - **借方、贷方和余额**：
     - 借方和贷方用于记录数量，并且必须成对出现。
     - 当前节点的借贷关系，必须与它所关联节点的借贷关系相互对应。
     - 余额的计算规则是：
       - 如果规则是 `DICD`，余额 = 借方 − 贷方。
       - 如果规则是 `DDCI`，余额 = 贷方 − 借方。

3. **示例操作**
   - 我们新建一条记录：按 `Ctrl + N`。
   - 关联到 `位置 - 3号箱`。
   - 借方输入 50，成本输入 100。
   - 然后使用 `Ctrl + J` 快捷键，可以直接跳转到关联的节点。
   - 返回物品版块，就能看到刚才录入的数据。

4. **修改规则**
   - 如果把所有位置节点的规则改成 `DDCI`，
   - 那么录入数据后，就可以从两个角度来查看：
     - 一方面，可以在 `3号箱` 节点中看到里面具体有什么。
     - 另一方面，可以在 `手机` 节点中看到它存放在哪个位置。

5. **搜索功能**
   - 最后，按 `Ctrl + F`，可以通过名称或者记录说明来快速查找节点。
