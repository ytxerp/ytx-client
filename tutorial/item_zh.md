# YTX 物品管理

大家好！今天我来介绍一下 **YTX** 如何管理物品。

在传统制造业中，了解材料的位置、品质以及是否能够快速获取，直接影响库存效率。**YTX** 针对材料难找的问题提供了解决方案，从而极大提升仓库效率。

当前软件版本：**0.2.5**

---

## 主界面

1. **登录**
   - 输入 **邮箱**、**密码** 和 **工区**（相当于公司名称）。
   - 可选择是否 **在本机保存密码**。

2. **菜单**
   - **菜单栏**：文件、编辑、帐户
   - **工具栏**：插入节点、追加节点、编辑节点名称、重置节点颜色、创建节点组、追加记录、删除、全选、反选、空选、账单、结算
   - **版块栏**：财务、物品、人脉、任务

3. **物品版块**
   - 也称为 **材料版块**、**产品版块** 或 **仓库版块**，统一称为 **物品**
   - 初始为空；结构通过 **节点** 来构建

---

## 核心属性

每个节点有 **三个关键属性**：

- **规则 (Rule)**：DICD 或 DDCI，用于决定叶节点数值的计算方式
- **类型 (Type)**：
  - **B（枝）**：组节点，用于统计叶节点数据
  - **L（叶）**：最终数据登记节点
- **单位 (Unit)**：
  - **内**：自有编号
  - **外**：映射编号
  - **空**：位置
  - 内部编号和位置最常用；外部编号用于映射关系

## 模拟操作

### 节点

1. **添加一个产品枝节点**
   - 快捷键：`Alt + P`
   - 名称：“苹果”，类型：“B”，单位：“内”

2. **在苹果下添加三个叶节点**
   - 选中苹果 → `Alt + P`
   - 名称：“手机”、“笔记本”、“台式机”，类型：“L”，单位：“内”

3. **在苹果下添加一个位置枝节点**
   - 选中苹果 → `Alt + N`（同层插入）
   - 名称：“位置”，类型：“B”，单位：空

4. **在位置下添加三个叶节点**
   - 选中位置 → `Alt + P`
   - 名称：“1号箱”、“2号箱”、“3号箱”，类型：“L”，单位：空

> 此时可以展开/折叠枝节点，或拖动节点。

**注意事项：**

- `Alt + P` 在当前选中的`枝节点`下追加子节点；若未选中任何节点，则在`root`下追加子节点，即生成界面所见的最顶层节点。
- `Alt + N` 在当前选中`节点`的同一层插入新节点；若未选中任何节点，则不执行操作。
- **类型和单位不可修改**；如果设置错误，只能删除节点重新添加。
- 枝节点只统计 **相同单位** 的叶节点数据。

---

### 记录

1. **添加一条记录**
   - 双击 `苹果 - 手机` → 打开记录页面
   - 工具栏会高亮可用操作

2. **关键属性**
   - **D**：可添加相关文档（仅保存路径，不存储文档本身）
   - **C**：盘库时使用
   - **关联**：可关联 **位置** 或 **其他产品**
     - 位置 → 表示存放在哪
     - 产品 → 表示进入下一生产步骤
   - **借方 / 贷方 / 余额**：
     - 借方和贷方登记数量
     - 借方和贷方一定是成对存在的，当前节点的**借方/贷方**数据一定和关联节点的**贷方/借方**数据一致
     - 余额计算方式取决于规则：
       - DICD：借方 − 贷方
       - DDCI：贷方 − 借方

3. **示例**
   - 插入记录：`Ctrl + N`
   - 关联：`位置 - 3号箱`
   - 借方：50，成本：100
   - 跳转到关联节点：`Ctrl + J`
   - 返回物品版块 → 可看到登记的数据

4. **修改规则**
   - 将所有位置节点修改为 **DDCI**
   - 数据录入完成后：
     - 可以通过 `3号箱` 查看其中有哪些物品
     - 可以通过 `手机` 查看存放位置

5. **搜索**
   - `Ctrl + F` 可以根据节点名或记录描述进行搜索
